# Struktur Website PietraMarmara

## Sitemap SEO-Ready

### Halaman Utama
- `/` - Homepage dengan hero section, kategori produk, layanan, testimoni
- `/produk` - Halaman daftar semua produk dengan filter
- `/produk/marmer` - Kategori marmer dan batu alam
- `/produk/tanaman-pangan` - Kategori tanaman pangan
- `/produk/tanaman-perkebunan` - Kategori tanaman perkebunan
- `/produk/buah-sayur` - Kategori buah dan sayuran
- `/produk/hasil-peternakan` - Kategori hasil peternakan
- `/produk/hasil-alam` - Kategori hasil alam lainnya

### Halaman Produk Individual
- `/produk/[slug]` - Detail produk individual

### Halaman Layanan
- `/layanan` - Halaman layanan utama
- `/layanan/custom-marble` - Custom marble design
- `/layanan/interior-consultation` - Konsultasi interior
- `/layanan/installation` - Instalasi proyek
- `/layanan/agricultural-partnership` - Kerjasama pertanian

### Halaman Informasi
- `/tentang` - Tentang PietraMarmara
- `/kontak` - Halaman kontak dengan form
- `/blog` - Blog untuk konten SEO

## Struktur Menu Navigasi

### Menu Utama
1. **Beranda** - Link ke homepage
2. **Produk** - Dropdown dengan 6 kategori utama
3. **Layanan** - Dropdown dengan 4 layanan utama
4. **Tentang Kami** - Informasi perusahaan
5. **Kontak** - Informasi kontak dan form

### Footer Menu
- Quick Links: Semua kategori produk
- Services: Semua layanan
- Contact: Informasi kontak lengkap
- Social Media: Facebook, Instagram, Twitter

- **Title**: "[Nama Produk] - PietraMarmara"

## Fitur Filter & Navigasi

### Filter Produk
- Filter by Category
- Filter by Price Range
- Filter by Material Type
- Filter by Application
- Sort by: Name, Price, Popularity, Newest

### Search Functionality
- Search by product name
- Search by category
- Search by material type
- Auto-suggest functionality

## Call-to-Action (CTA) Strategy

### Primary CTA
- "Request Quote" - Untuk produk marmer
- "Konsultasi Proyek" - Untuk layanan custom
- "Hubungi Kami" - Untuk informasi umum

### Secondary CTA
- "Eksplorasi Produk" - Untuk browsing
- "Lihat Detail" - Untuk produk individual
- "Download Katalog" - Untuk marketing material

## Responsive Design Breakpoints

- **Mobile**: < 768px
- **Tablet**: 768px - 1024px
- **Desktop**: > 1024px

## Performance Optimization

- Image optimization dengan Next.js Image
- Lazy loading untuk gambar produk
- Code splitting untuk halaman
- Caching strategy untuk produk data
